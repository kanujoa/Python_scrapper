<beautifulsoup 설치 방법>
replit에서 큐브 모양(packages)를 클릭 후 beautifulsoup을 검색하고 beautifulsoup 버전4를 설치한다. (+ 버튼을 누르면 설치된다.)
requests 패키지도 함께 설치되어 있는지 확인한다.(packages에서 검색창에 아무것도 입력하지 않은 상태를 만들면 설치된 패키지들이 뜬다.)
requests를 이용하면 웹사이트 호출하는 것을 쉽게 해 줄 것이다.

모든 웹사이트가 스크래핑 가능한 것은 절대 아니다.


* VSCode 같은 곳에서 터미널을 이용하여 설치하는 법
pip install beautifulsoup4
pip install requests
