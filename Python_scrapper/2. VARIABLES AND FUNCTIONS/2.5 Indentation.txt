1. 파이썬에서는 해당 코드가 어떤 것 안에 들어있다는 것을 쉽게 알게 해 주기 위해 들여쓰기를 한다.
탭 키를 이용하거나 스페이스바를 이용해 2칸, 4칸 이런 식으로 들여쓰기를 하면 된다.
(다른 프로그래밍 언어에서는 들여쓰기가 존재하지 않는다. 다른 언어에서는 함수를 만들 때 항상 중괄호를
쓰고 중괄호 안에 있는 것들은 전부 function안에 있는 것으로 취급한다.)
def say_hello():
  print("hello how r u?")     --> 들여쓰기를 하니 어떤 print문이 어떤 function에 속해 있는지를
                                          직관적으로 알아볼 수 있다.
def say_bye():
  print("bye bye")

say_hello()      --> 만약 이 코드를 들여쓰기하여 작성하면 say_hello는 say_bye 함수 안에서 print문과
                         함께 실행되는 함수가 될 것이다.
또한, 함수를 정의했으면 그에 속하는 실행문을 작성하는 것은 필수이다.
